# Odysseus Companion Support

Odysseus Companion is a private client for pairing with an Odysseus server,
continuing chat sessions, and running signed companion commands from a trusted
device.

## Getting help

For support, open an issue in this repository:

https://github.com/jongan69/odysseus-expo/issues

Include:

- Your app version
- Your iOS version and device model
- Whether you paired by QR code or manual payload
- The visible error message, if any
- Whether the Odysseus server is on the same local network or an HTTPS origin

Do not include companion access tokens, signing keys, private prompts, or server
secrets in public issues.

## Pairing requirements

The app requires an Odysseus server with the companion API enabled. Pairing uses
an admin-generated payload containing the server host, port, and companion
token.
